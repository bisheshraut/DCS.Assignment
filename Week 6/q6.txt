someAlgorithm(int n)
{
   Input
   for(i=0; i<n; i++)
   {
     Assignment
     for(j=0; j<n; j++)
       Comparison
   }
   Arithmetic operation
   Output
}
In this code 
for the first loop :
for(i=0; i<n; i++) this loop runs n times : So the time complexity is O(n).
for the nested loop :
for(j=0; j<n; j++) the loop runs n times :So the time complexity is O(n).
As the second loop is nested loop the time complexity of the algorithm becomes : O(n*n)=O(n^2).
